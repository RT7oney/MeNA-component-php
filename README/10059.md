# 获取商户导入用户积分记录(父表)
## 监听端口号
##### *10059*
## 接口传入参数（必须基础参数）
##### **商户id** : *merchant_id*
##### **查询开始日期(时间戳)** : *start_date* 可选
##### **查询结束日期(时间戳)** : *end_date* 可选
##### **当前页数** : *page* 可选
##### **每页多少条** : *limit* 可选


## 成功返回
##### **code=>10059001,msg=>'成功','data' => 表信息**

## 失败返回
##### **code=>10059002,'msg'=>'参数不合法'**
##### **code=>10059003,'msg'=>'缺少必要参数'**
##### **code=>10059004,'msg'=>'商户id格式不合法'**
##### **code=>10059005,'msg'=>'商户不存在'**
##### **code=>10059006,'msg'=>'无数据'**
