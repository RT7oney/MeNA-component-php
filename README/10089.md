# 增加关联会员卡
## 监听端口号
##### *10089*
## 接口传入参数（必须基础参数）
##### **用户uuid** : *uuid*
##### **商户名称** : *merchant_name*
##### **卡号或手机** : *card_no*

## 成功返回
##### **code=>10089001,msg=>'关联会员卡成功'**


## 失败返回
##### **code=>10089002,'msg'=>'参数不合法'**
##### **code=>10089003,'msg'=>'参数不合法'**
##### **code=>10089004,'msg'=>'用户不存在'**
##### **code=>10089005,'msg'=>'此会员卡已关联'**
##### **code=>10089006,'msg'=>'系统异常'**
