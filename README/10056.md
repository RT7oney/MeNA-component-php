# 获取代理商下所有商户
## 监听端口号
##### *10056*
## 接口传入参数（必须基础参数）
##### **代理商id** : *agent_id*
##### **当前页数** : *page*
##### **每页多少条** : *limit*
##### **排序字段** : *sortkey*
##### **排序** : *sort*


## 成功返回
##### **code=>10056001,msg=>'查询成功','data'=>'代理商商户信息'**

## 失败返回
##### **code=>10056002,'msg'=>'无数据'**
##### **code=>10056003,'msg'=>'代理商不存在'**
##### **code=>10056004,'msg'=>'代理商id格式不合法'**
##### **code=>10056005,'msg'=>'缺少必要参数'**
##### **code=>50056001,'msg'=>'参数不合法'**
