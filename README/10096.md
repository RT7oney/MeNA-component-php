# 获取小金消息
## 监听端口号
##### *10096*
## 接口传入参数（必须基础参数）
##### **用户uuid** : *uuid*
##### **查询时间戳** : *search_time*


## 成功返回
##### **code=>10096001,msg=>'操作成功','data'=>数据信息**
##### **code=>10096001,msg=>'无数据','data'=>数据信息**


## 失败返回
##### **code=>10096002,'msg'=>'参数不合法'**
##### **code=>10096003,'msg'=>'用户不存在'**
##### **code=>10096004,'msg'=>'系统错误'**
##### **code=>10096005,'msg'=>'系统错误'**
##### **code=>10096006,'msg'=>'参数不合法'**
