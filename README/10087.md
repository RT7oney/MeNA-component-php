# 查看赠送记录详细信息
## 监听端口号
##### *10087*
## 接口传入参数（必须基础参数）
##### **支出记录ID** : *id*
##### **赠送用户uuid或则获得者uuid** : *uuid*
##### **创建时间戳** : *create_time*

## 成功返回
##### **code=>1008701,msg=>'操作成功',data=>数据信息**


## 失败返回
##### **code=>1008702,'msg'=>'该记录不存在'**
##### **code=>1008703,'msg'=>'用户不存在'**
##### **code=>1008704,'msg'=>'该记录表不存在'**
##### **code=>1008705,'msg'=>'参数id、create_time、uuid不合法'**
##### **code=>5008701,'msg'=>'参数不合法'**
