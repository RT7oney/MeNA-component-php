# 商户获取已加入积分兑换计划的用户
## 监听端口号
##### *10067*
## 接口传入参数（*）为必须参数
##### **商户id（*）**: *merchant_id*
## 成功返回
##### **code=>10067001,msg=>'成功',data=>手机号**
## 失败返回
##### **code=>10067006,'msg'=>'系统异常'**
##### **code=>10067005,'msg'=>'系统异常'**
##### **code=>10067004,'msg'=>'商户不存在'**
##### **code=>10067003,'msg'=>'商户id格式不合法'**
##### **code=>10067002,'msg'=>'参数不合法'**
