# 根据虚拟卡号码获取取用户姓名手机号
## 监听端口号
##### *10044*
## 接口传入参数（必须基础参数）
##### **虚拟卡卡号** : *invit*
## 成功返回
##### **code=>10044001,msg=>'成功',data=>'用户姓名手机号信息'**
## 失败返回
##### **code=>10044002,'msg'=>'参数不合法'**
##### **code=>10044003,'msg'=>'虚拟卡参数非法'**
##### **code=>10044004,'msg'=>'虚拟卡号不存在'**
##### **code=>10044005,'msg'=>'用户表异常'**
##### **code=>10044006,'msg'=>'用户异常'**
