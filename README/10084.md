# 获取银行卡类型
## 监听端口号
##### *10084*
## 接口传入参数（必须基础参数）
##### **银行卡卡号** : *bankcard*


## 成功返回
##### **code=>10084001,msg=>'操作成功',data=>数据信息**


## 失败返回
##### **code=>10084002,'msg'=>'未知的银行卡号码'**
##### **code=>10084003,'msg'=>'系统错误'**
##### **code=>10084004,'msg'=>'银行卡号码错误'**
##### **code=>10084005,'msg'=>'参数不合法'**
