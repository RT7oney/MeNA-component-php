# 删除关联会员卡
## 监听端口号
##### *10098*
## 接口传入参数（必须基础参数）
##### **用户uuid** : *uuid*
##### **会员卡id** : *card_id*


## 成功返回
##### **code=>10098001,msg=>'删除成功'**

## 失败返回
##### **code=>10098002,'msg'=>'系统异常'**
##### **code=>10098003,'msg'=>'此会员卡不存在'**
##### **code=>10098004,'msg'=>'用户不存在'**
##### **code=>10098005,'msg'=>'参数不合法'**
##### **code=>10098006,'msg'=>'参数不合法'**
