# 增加我的心愿单
## 监听端口号
##### *10092*
## 接口传入参数（必须基础参数）
##### **赠送用户uuid或则获得者uuid** : *uuid*
##### **商户名称** : *merchant_name*
##### **地区** : *area*

## 成功返回
##### **code=>10092001,msg=>'添加心愿单成功!'**


## 失败返回
##### **code=>10092002,'msg'=>'参数不合法'**
##### **code=>10092003,'msg'=>'参数不合法'**
##### **code=>10092004,'msg'=>'用户不存在'**
##### **code=>10092005,'msg'=>'此心愿单已添加!'**
##### **code=>10092006,'msg'=>'系统异常'**
